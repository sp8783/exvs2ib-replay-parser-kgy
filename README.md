# EXVS2IB Tracker (KGY専用)

機動戦士ガンダム エクストリームバーサス2 インフィニットブースト（EXVS2IB）の対戦履歴・勝敗を自動で記録する、**KGY専用**の戦績管理ツールです。

## 概要
YouTubeなどに投稿された対戦動画から、画面上の特定のテキストをOCRで解析し、試合ごとの結果（勝敗・機体など）を自動で抽出・記録します。
最終的には、リアルタイムでブラウザ再生中の動画を解析できるようにする予定です。

現在は以下のワークフローで動作します：

1. 対象の対戦動画をダウンロード  
2. フレームごとに画像を切り出し  
3. OCRで試合結果を解析  
4. CSVやDBに記録
